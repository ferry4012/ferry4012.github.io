  2. 在文件夹[x]下就会有master和hexo两个分支，默认是master分支。用git checkout 命令进行分支切换。

  3. 解压到另外的文件夹[xxx]下，并在[xxx]下做需要的修改、新增文章、发布等操作。可直接发布到githubpages里。

  4. 将[xxx]下public等无用文件删除，并压缩为zip包，然后放回到[sourceOfHexo]文件夹下。
